## SyntaxError 練習問題

1. **問題:** `print("Hello"`
   **修正後:** `print("Hello")`

2. **問題:**

   ```python
   if x == 10 print(x)
   ```

   **修正後:**

   ```python
   if x == 10:
       print(x)
   ```

3. **問題:**

   ```python
   if True:
   print("OK")
   ```

   **修正後:**

   ```python
   if True:
       print("OK")
   ```

4. **問題:** `message = "Hello`
   **修正後:** `message = "Hello"`

5. **問題:**

   ```python
   class = "Math"
   ```

   **修正後:**

   ```python
   subject = "Math"
   ```

6. **問題:**

   ```python
   def greet:
       print("Hi")
   ```

   **修正後:**

   ```python
   def greet():
       print("Hi")
   ```

7. **問題:** `print('Alice's cat')`
   **修正後:** `print("Alice's cat")`

8. **問題:** `numbers = [1, 2,, 3]`
   **修正後:** `numbers = [1, 2, 3]`

9. **問題:** `print "Hello"`
   **修正後:** `print("Hello")`

10. **問題:**

    ```python
    if x = 1:
        pass
    ```

    **修正後:**

    ```python
    if x == 1:
        pass
    ```
